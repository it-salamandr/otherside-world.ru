# otherside-world.ru

Перед вами архив сайта о самопознании, изменении мышления и развитии внутренних возможностей человека.

В нём собраны авторские программы:

- **«Бдилка»** — для работы с осознанностью, вниманием и созданием новых жизненных сценариев.
- **«Перелом»** — для анализа и изменения привычных моделей мышления и поведения.
- **«Схемокодер»** — для развития навыков и мышления через игровую систему программирования.
- **«Артефакт: Изнанка Мира»** — о выборе, трансформации и создании собственного пути.
